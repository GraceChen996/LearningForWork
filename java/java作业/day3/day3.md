**5.ABDG**

静态方法是属于类本身的，而不是属于类的实例，静态方法随着类的加载而加载，此时泛型还没传递真正的参数，因此在静态方法中无法直接引用泛型类型 T。所以 

```
static T y;
static T s1() {return null;}
static void s2(T i) {}
```

这种形式都是错误的。如果静态方法需要使用泛型参数，可以将泛型参数定义在方法上。
